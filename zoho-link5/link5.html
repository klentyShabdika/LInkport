How to prevent Duplicates on Linkport-Zoho?
When you import a lead from LinkedIn to Zoho CRM using Linkport, the “Add to Zoho” option next to the leads’ name changes to “Added to Zoho”.

The “Added to Zoho” doesn’t appear for a lead that you have previously added if you have not mapped the LinkedIn profile to a field in Zoho.

Here is how you can map the LinkedIn profile to a field in Zoho CRM,

  1. In Zoho CRM, go to Settings → Setup - Customization - Modules & Fields.


  2. Under Modules, Go to Leads → Fields → “Create + Edit Fields” Button. You will find all your existing fields in the Create page/tab.



  3. Under “New Fields”, drag the “Single Line” field.
    • For eg., we have named the single line field as “LinkedIn Profile URL”, so we can map the LinkedIn profile to a field in Zoho.
    • Click on “Save” or “Save & Close” so it gets updated on Linkport - Set Up Account page.
